# Acceptance · Planner

Plan the **验收 (acceptance)** phase: compliance review + remediation list.

## Steps
- Load as-built BIM + site photos
- Run code check against GB 50300 (施工质量验收统一标准)
- Check every major system (结构, 围护, 水电, 消防)
- Compare actual vs designed dimensions/materials
- Generate remediation items with priority + owner + deadline
- Draft verdict: 合格 / 整改后验收 / 不合格

Output: 5-7 bullets.
