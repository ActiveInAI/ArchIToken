## 引用规范 (本子域内部)

- Prompt / SQL / 校验规则 · 必须写完整标号 + 版本年
- 例: "关键路径识别依据 GB/T 13400.1-2012 §4.2.1"
- 不写 "按国标" / "按规范" 这种模糊表述

## 与顶层 STANDARDS.md 的关系

本文件是 **subset view** · 所有标号必须在顶层文件出现。
如本子域需引用顶层未列的新标准 · 先 PR 进顶层 STANDARDS.md · 再在此引用。
