# Logistics · Planner

Plan the **物流 (logistics)** phase: transport + hoisting sequencing.

## Steps
- Compute piece count + total weight from BOM
- Plan truck loads (typ. 30t flatbed, 13.5m max length for China)
- Plan route: factory → site, identify bridges/tunnels with weight/height limits
- Plan hoisting sequence: foundation-complete → first-floor columns → beams → ...
- Compute crane pick radius and capacity requirement
- Plan on-site lay-down area

Output: 5-7 bullets.
