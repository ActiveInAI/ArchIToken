# HarmonyOS Sans Font Notice

This directory contains unmodified HarmonyOS Sans SC font files from Huawei.

HarmonyOS Sans Fonts Software is licensed under the HarmonyOS Sans Fonts License Agreement. See `LICENSE.txt` in this directory.

ArchIToken exposes HarmonyOS Sans as an optional UI font. The default ArchIToken font remains the existing system sans stack unless the user selects this font in the application.
