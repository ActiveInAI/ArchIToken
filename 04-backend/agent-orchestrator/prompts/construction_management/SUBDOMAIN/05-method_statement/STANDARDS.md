## 3. 超规模判定基准 (建办质〔2018〕31 号 附件二)

| 工程类别 | 超规模阈值(挑战需专家论证) |
|---|---|
| 深基坑 | 开挖深度 ≥ 5m(或周边地下管线等) |
| 高大模板 | 搭设高度 ≥ 8m 或跨度 ≥ 18m |
| 起重吊装 | 单件 ≥ 100kN(约 10t) 或 塔吊自高 ≥ 30m |
| 脚手架 | 落地 ≥ 24m / 悬挑 ≥ 20m / 附着升降 |
| 拆除 | 城区繁华区 · 人员密集 · 装配房 · 5 类以上 |
| 暗挖 | 所有暗挖 |
| 水下作业 | 所有水下 |
| 沉井 / 顶管 | 所有 |
