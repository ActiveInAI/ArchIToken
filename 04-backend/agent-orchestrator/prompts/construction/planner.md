# Construction · Planner

Plan the **施工 (construction)** phase: 4D simulation + crew scheduling.

## Steps
- Build WBS (work breakdown) from BIM + fab plan
- Estimate crew size per task (regional labor norm)
- Compute critical path; set milestones
- Plan daily safety briefing topics
- Plan QA/QC hold points
- Plan weather / force-majeure buffer

Output: 5-7 bullets.
